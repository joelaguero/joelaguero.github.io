Nextdoor

I've been at Nextdoor for about 2 months now. In this time, I've successfully worked with four engineers and one product manager to launch automatic link previews for new posts, which has been rolled out platform-wide. Since successfully completing this project, I've expanded my responsibilities to include three other projects. One involves designing a completely new type of post on Nextdoor, another will introduce a much-needed improvement to the existing post creation experience, and the third will be introduce a bit of personality in an area that may often go unnoticed. Throughout these projects, I've used the following tools:

  • Experience diagramming
  • User interaction flows
  • Storyboards
  • Thumbnail sketches
  • Wireframes
  • Mock-ups
  • InVision prototypes
  • Interaction animations
  • Project-specific design specs
  • HTML5 & CSS3

Smartlinks
